At Campana & Schott, I built an AI-powered lead generation agent that automated what was previously hours of manual research. Here's how it works.

## Architecture

The system follows a pipeline pattern:

1. **Discovery** — Browser automation scrapes public business directories
2. **Enrichment** — LLM extracts structured data from unstructured pages
3. **Scoring** — Custom scoring model ranks leads by relevance
4. **Delivery** — Results pushed to CRM via API

## Browser Automation

We used Playwright for the browser automation layer. It handles JavaScript-rendered pages, CAPTCHAs (with human-in-the-loop fallback), and rate limiting out of the box.

```python
async def discover_leads(query: str, max_pages: int = 10):
    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=True)
        page = await browser.new_page()
        results = []
        for page_num in range(max_pages):
            await page.goto(f"{BASE_URL}/search?q={query}&p={page_num}")
            results.extend(await extract_listings(page))
        await browser.close()
    return results
```

## LLM Integration

The enrichment step uses structured output from an LLM to turn raw HTML into clean, typed lead records. The key is prompt engineering with strict output schemas.

## Results

The agent reduced lead research time from ~4 hours to ~15 minutes per batch, with comparable accuracy to manual research.
